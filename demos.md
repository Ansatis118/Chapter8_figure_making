# Real-World Demos and Figure Scripts

The design theory, patterns, and guidance in this skill are based on common publication-quality scientific visualization practices. Those scripts are the canonical demos for this skill.

**Repository:** [https://github.com/Ansatis118/Chapter8_figure_making](https://github.com/Ansatis118/Chapter8_figure_making)

## Project folders (demos)

Each folder contains Python scripts and (in `figures/`) outputs that follow the skill's consistent publication style. Use them as references for bar charts, trend plots, heatmaps, and multi-panel layouts.
